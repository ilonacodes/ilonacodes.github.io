Online Fashion Shopping App

UI/UX Case Study

Design Process

Step 1. Observation

Abstract

The iLookStore is a fashion retail shopping app. The app allows to accomplish the following:

  • Find suitable item within the category
  • Filter clothing items by size, colour, etc.
  • See clothing item details
  • Save item for later
  • Store personal data, so that it is easy to buy items again
  • Buy found items

Problem Assumptions

  1. Traditional shopping takes lots of time
  2. Not all items are available for purchasing from the catalog
  3. Hard to find a certain item if all clothing items are badly categorised and badly sorted

User Interviews

To validate the problems:

Interviews with contacts from friends and colleagues

To understand:

Their needs, goals and pains

User Scenario & stories

User Story

Filter list of items by size


As Alice,

I want to filter items in the shopping list by size

so that I can find the item of suitable size

Acceptance Criteria

Given I am on the shopping list screen

When I tap on the filter icon

Then I see the filter screen with a size filter

When I choose a size “36”

And I come back to a list of clothing items

Then I see only the items that have size “36” in the stock
